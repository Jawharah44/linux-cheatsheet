# Linux Cheatsheet 🐧 
This repository will grow over time. Notes are added as I learn and practice.

The real power of Linux comes from its command-line interface, called the **shell**.

## Linux Commands
A Linux command usually consists of:
- a **program name**
- followed by **options**
- and **arguments**

Options usually begin with `-` (short options) or `--` (long options).

### Example
```bash
ls -l /home/jawharah
```
ls -> program
-l ->  option (long listing format)
/home/jawharah → argument (the target)
